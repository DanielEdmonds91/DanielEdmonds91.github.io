arrays and objects

Arrays and objects are very similar in that they both contain multiples of information.
The main difference is that arrays act as a series of containers that function as a whole.
Having a single name but with the ability to store things within the subcompartments named elements . Usually arrays are used to store the same type of thing

Objects are used to store different types of things while under a common identifier. Like arrays they still store multiple things but the difference being the type fo things can be different. It is possible to store an array within an object.

Functions

Functions are one of the most important building blocks of javascript. A functions is basically a set of statements that performs a task and/or calculates a value.
The use of functions comes from their ability to run the same piece of code without repeating it. So you can say if “a” conditions are met perform “x” function but “x” function could be used if “b” conditions are met also.

The DOM

The DOM is the acronym for “Document Object Model” its the method that brings all the separate technologies together on a web page. It lets the html and css interact and can change what we see. The DOM translates all the parts of the HTML into objects within the webpage .
These objects can be changed events can be written which allows for the parts of the webpage to become more dynamic. To interact with the DOM a developer may use the inspect element of a web page which allows him to play with the page in the way they see fit.
