What are the differences between relative, absolute, and fixed positioning?

When talking about the different css properties for css its important to put things in a simple way for the reader to understand so I will be using my words to the best of my ability along with images to help demonstrate how positioning works in css

relative positioning

Using relative positioning allows us to change the top,right,left, and bottom element relative to where it would be in the default or "static" postioning in html
in this example is will use relative postioning to push the child one element 10 pixels to the left relative to where it was for example

This is an uncommon way to position elements because it removes the element out of the document slow and makes it more difficult so position the other elements around it

Absolute positioning

Absolute positioning positions an element without regard to any other elements . It is used to position elements anywhere on the page and not have any other elements interact with it

In this image the child one element has had an absolute position applied to it. it looks like the child two element has been deleted but in actuality child two is sitting underneath child one
as child one has been pulled out of the document flow and had its position placed detached from the other elements.

Fixed positioning

Fixed positioning is similar to absolute positioning with the difference being its position is is in reference to the entire html document
it also stays in that position of the page no matter where the user scrolls

In this image the child one element has been set to a fixed position.
it is placed over the child three element because i have scrolled down slightly and instead of the elements staying in its position it following the position as i scrolled overlapping the child three element.
